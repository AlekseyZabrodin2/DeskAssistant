### DeskAssistant

This project README is available in Russian: [README.ru.md](README.ru.md)

DeskAssistant is an application that provides a set of tools for everyday work. At the moment, it includes a tool that automatically recognizes a list of colleaguesâ€™ birthdays from an image, lets you review and edit the results, and sends notifications to colleagues.

### Features

- Upload an image containing a list of birthdays
- Recognize text using OCR
- Edit the list before sending
- Detect the names of birthday people
- Send notifications to colleagues

### Planned features

- Add new productivity tools
- Calendar integration
- Improved recognition algorithms

### Technologies

- .NET 9
- OCR (Tesseract / Emgu CV)
